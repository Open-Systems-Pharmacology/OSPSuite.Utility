# OSPSuite.Utility

This component provides some cross-cutting concerns implementation used in the Open Systems Pharmacology Project such as:

* Event publishing
* File locking
* Logging
* Validation
* Dynamic visitor implementation
* Useful extension methods
* ... and much more ...
